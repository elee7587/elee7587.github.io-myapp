Welcome to the Restaurant Hygiene Rating

This web app allows you to upload a dataset of restaurant reviews and related information in CSV format. We require certain information that will be specified below. This web app will provide insights into restaurant hygiene ratings based on customer reviews. It will analyze the data using a machine learning model to predict restaurant's outcomes based on review sentiments and other features. As this is a supervised machine learning model, the model will need previously labeled data in order for this model to be successful. Stated below will be the specifications required for this to work.

Specifications

Each CSV file should contain the following columns: (In order for this to be functional, the column names need to be the same as shared below. )

  • review: Text of the customer review.
  • label: Sentiment label (e.g. Training: 1, 0 Testing: [None])
  • restaurant: Restaurant Name.
Optional Columns The following columns are optional but can enhance the analysis:
  • cuisines: Type of cuisine offered by the restaurant.
  • avg_rating: Average rating of the restaurant.
  • zip_code: Location of the restaurant.

Who should use this?
Researchers, restaurant owners, or anyone interested in analyzing review data to gain insights on customer sentiments.

How to use:
1. Prepare your CSV file with columns: review, label, cuisines, and avg_rating.
2. Choose the CSV file using the file picker below.
3. Click "Upload and Process" to see the model predictions.
