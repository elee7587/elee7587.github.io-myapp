Welcome to the Data Mining Application

This web app allows you to upload a dataset of restaurant reviews and related information in CSV format. It will analyze the data using a machine learning model to predict outcomes based on review sentiments and other features.

Who should use this?
Researchers, restaurant owners, or anyone interested in analyzing review data to gain insights on customer sentiments.

How to use:
1. Prepare your CSV file with columns: review, label, cuisines, and avg_rating.
2. Choose the CSV file using the file picker below.
3. Click "Upload and Process" to see the model predictions.
